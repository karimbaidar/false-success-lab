# Clean scanner redesign

This replacement redesign removes the noisy three-column cockpit layout from the previous UI pass.

The first viewport now focuses on one thing:

- a short marketing headline
- one public GitHub repo input
- one scan button
- a small backend status

The multi-agent animation and report card appear only after the user starts a scan. Local import and built-in scenarios remain available below the primary scanner instead of competing with it on first load.
